## 插件简介

当bot在群里遭遇频繁禁言、群友发言对bot有明显敌意、排斥甚至辱骂bot时，插件会帮助bot识别并退群，保护bot身心健康，从源头上节省Tokens。


### 🛡️ 三种退群触发机制

- **禁言退群** — bot被群管反复禁言达到阈值后自动退群，支持按次数或按累计时长触发，并在退群前通过修改群名片发出倒计时预警。
- **LLM 语义判定退群** — 当bot参与对话后，通过关键词初筛 + 大语言模型二次判定的双层过滤机制，识别针对bot的敌意发言，累计达标后自动退群；进入警戒线后还会在群内发送提示消息。
- **指令退群** — 管理员可直接通过指令让bot当场离开指定群聊。


### 💡 插件特点

- **双层过滤**：关键词初筛 + LLM 二次判定，敌意判定仅在bot参与对话后触发，节省 Token 开销
- **防注入保护**：用户消息隔离，防止恶意用户利用 Prompt 注入操控bot
- **宕机兜底**：即使在禁言等待退群期间程序重启，插件也会自动补发退群任务
- **白名单豁免**：白名单群聊对一切退群机制完全免疫


## 📦 安装和使用

在 AstrBot 插件市场搜索 **astrbot_plugin_bye**，点击安装即可生效启用。  
前往 AstrBot 插件配置面板即可进行设置。

---

## ⌨️ 命令一览
| 命令 | 说明 | 权限 |
|------|------|------|
| `/bye` | 让bot发送告别语并立即退群 | 所有人 |
| *自定义关键词* | 在控制面板设定自定义退群触发词，不带前缀也可触发 | 所有人 |
| `/bye_stats` | 查看各群对bot的不友善对待情况统计报告 | 仅bot管理员 |
| `/bye_clear` + 群号（可不指定） | 清理不友善记录，可指定群号或不指定群号（一键清空所有） | 仅bot管理员 |
*注：命令前缀依照Astrbot设定，本表以Astrbot默认的`/`为例，如设置成`!`，则命令变为`!bye`、`!bye_stats`、`!bye_clear`

## 📋 配置项详解

### 【全局设置】

| 配置项 | 说明 |
|-------|------|
| 🛡️ 群聊白名单 | 白名单内的群聊对插件完全免疫，所有退群机制均不生效 |
| 💬 告别语 | 触发退群前向群内发送的最后一条消息 |

### 【指令退群】

| 配置项 | 说明 |
|-------|------|
| 🛠️ 退群指令开关 | 控制 `/bye` 指令的启用或禁用 |
| ⌨️ 自定义退群关键词 | 自定义无前缀的退群触发词（如"你褪裙吧"） |

### 【禁言退群】

| 配置项 | 说明 |
|-------|------|
| 🛑 禁言次数阈值 | 累计被禁言达到该次数后自动退群（0 = 不启用） |
| 🛑 禁言时长阈值 | 累计被禁言达到该时长（小时）后自动退群（0 = 不启用） |
| ⏱️ 禁言时长计算方式 | 可选"按判罚时间"或"按实际禁言时间"计算累计时长 |
| ↩️ 解除禁言后补回计数 | 群管提前解禁时是否扣回本次禁言次数的案底 |
| ⚠️ 剩余禁言次数提示 | 进入倒计时时自动修改群名片追加退群预警后缀 |
| ⚠️ 剩余禁言时长提示 | 剩余允许时长进入警戒线时触发名片预警 |

### 【LLM 语义判定退群】

| 配置项 | 说明 |
|-------|------|
| 🤬 触发关键词 | 命中关键词的对话才会送入 LLM 做敌意判定（列表留空则关闭） |
| 🧠 调用模型 ID | 可指定专用的轻量模型做判定，留空则使用当前主聊天模型 |
| 🤬 敌意发言容忍次数 | 被 LLM 判定为敌意的累计次数达标后触发退群 |
| ⚠️ 剩余不友善发言容忍次数提示 | 进入倒计时时向群内直接发送警告消息（0 = 关闭） |
| 📢 不友善退群提示消息模板 | 警告消息模板，支持 `{count}`、`{max}`、`{left}` 变量 |
| 意图判定 Prompt | 可自定义 LLM 敌意判定提示词（内置 Markdown 编辑器） |
